# outputs/final_results

Sometimes I dump **intermediate comparison tables** here inside `outputs/` before I’m ready to bless them as “real” artifacts.

## Why it exists

The top-level **`final_results/`** folder is what I treat as publication-ready. This nested copy is just **convenient staging** when a script defaults its output path under `outputs/`.

## What I do

If I see something valuable here, I **copy or symlink** it up to `/final_results/eval/` so I’m not confused later about which CSV is canonical.
