# old_versions

I’m keeping this around as a **museum of earlier pipeline layouts**—old stages, datasets, and scripts that the *current* tree under `src/` / `utilities/` / `agents/` no longer imports directly.

## What’s inside

- **`stage_1/` … `stage_5/`** — Incremental snapshots from when I was iterating on training stacks.
- **`old/`**, **`archive/`**, **`data/`** — Legacy assets (videos, YOLO exports, notebooks) I didn’t want to delete outright.
- **Per-stage READMEs** — Some subfolders already explain their own experiment era.

## How I use it today

Honestly? Mostly for **diffing or grabbing a one-off script**. If I’m doing new work, I stay in the reorganized root (`README.md` at repo top). I fall back to `old_versions/` when I need to prove to myself how I used to compute something.
