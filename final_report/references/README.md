# final_report/references

I hoard **bibliography fodder** here—PDFs, saved HTML pages, and my own index markdown so I remember where I downloaded what.

## What’s inside

- **Category folders** — e.g. classic RL papers, SB3 docs, Gymnasium API notes, etc.
- **`download_index.md`**, **`reference_collection.md`** — My personal maps of the tree.
- **`Source_Code_To_Analyze/`** *(if present)* — Optional vendored repos for close reading; I don’t import this into Python code.

## Heads-up

This can get huge. I don’t assume every teammate clones all PDFs—treat it as an archive, not a runtime dependency.
