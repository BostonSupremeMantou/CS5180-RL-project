# final_report/drafts

I use this corner for **work-in-progress prose**—outlines, checklists, literature-review notes, anything that isn’t the final LaTeX yet.

## What you might see

- Markdown files with rough section text.
- Grading or instruction copies from the course (sometimes gitignored so they don’t clutter the repo).

## How I use it

I draft here first, then paste polished paragraphs into `final_report.tex`. Nothing in this folder is required to build the PDF unless I explicitly `\input` it (I usually don’t).
